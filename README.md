# Exercícios em Java

Este repositório contém exercícios desenvolvidos durante meus estudos em Java,
com foco em lógica de programação, fundamentos da linguagem e boas práticas.

## Estrutura do repositório
- Cada pasta representa um exercício ou dia de estudo
- Os exercícios seguem uma progressão de dificuldade
- Cada exercício possui seu próprio README com explicações

## Conteúdos abordados
- Estrutura básica do Java
- Entrada e saída de dados
- Condicionais
- Laços de repetição
- Arrays e listas
- Lógica de programação

## Objetivo
Consolidar os fundamentos de Java e criar um histórico de evolução para fins de estudo e portfólio.
