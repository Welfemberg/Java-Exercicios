# Dia 02 - Entrada de dados com Scanner

Exercício em Java focado na leitura de dados do usuário por meio da classe Scanner,
trabalhando com diferentes tipos de entrada e saída formatada.

## Objetivo
- Ler dados digitados pelo usuário
- Trabalhar com entrada de dados do tipo String e int
- Exibir mensagens personalizadas no console
- Realizar cálculos simples com base nos dados informados

## Funcionalidades
- Solicita o nome do usuário
- Solicita a idade do usuário
- Solicita o ano em que o usuário está respondendo o questionário
- Exibe uma mensagem personalizada com nome e idade
- Calcula e exibe a idade que o usuário terá no ano de 2050

## Conceitos utilizados
- Importação de classes (`import`)
- Classe Scanner
- Entrada de dados pelo teclado
- Tipos primitivos (`int`)
- Strings
- Constantes (`final`)
- Operações aritméticas
- `System.out.println`
- `System.out.printf`
- Comentários no código

## Observação
Este exercício faz parte de uma sequência de estudos em Java, seguindo uma progressão
de dificuldade com foco em lógica de programação e fundamentos da linguagem.
