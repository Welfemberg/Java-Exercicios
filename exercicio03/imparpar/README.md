# Dia 03 - Verificador de número par ou ímpar

Exercício em Java com foco no uso de estruturas condicionais para tomada de decisão
com base em dados informados pelo usuário.

## Objetivo
- Ler um número inteiro digitado pelo usuário
- Verificar se o número é par ou ímpar
- Exibir o resultado no console

## Funcionalidades
- Solicita um número inteiro ao usuário
- Verifica se o número é par ou ímpar utilizando o operador módulo (`%`)
- Exibe uma mensagem informando o resultado

## Conceitos utilizados
- Estrutura condicional (`if / else`)
- Operador módulo (`%`)
- Comparações lógicas
- Classe Scanner
- Entrada de dados pelo teclado
- `System.out.println`
- Comentários no código

## Observação
Este exercício faz parte de uma sequência de estudos em Java, seguindo uma progressão
de dificuldade com foco em lógica de programação e fundamentos da linguagem.
